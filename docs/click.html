on_key_down(keycode, modifiers)

Override to receive keydown events.

Parameters:

  • keycode – [ascii-code, key] ascii-code is an int, and key is a string. Example: [49, ‘1’] when the 1 key is pressed.

  • modifiers – a list of held-down modifier keys, like ‘shift’, ‘ctrl’, ‘alt’, ‘meta’

on_update()

Override to get called every graphics frame update, typically around 60 times per second.
