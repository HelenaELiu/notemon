synth.py

class imslib.synth.Synth(filepath=None, gain=0.8)

Generator that creates sounds from a FluidSynth synthesizer bank.

Parameters:

  • filepath – Path to the file containing the synthesizer bank. If None, Synth will load a locally cahced FluidR3_GM.sf2 file. If uncached, Synth will download FluidR3_GMsf2.

  • gain – The gain, a float between 0 and 1.

cc(chan, ctrl, val)

Send control change value. The controls that are recognized are dependent on the SoundFont. Typical values include -

1 : vibrato
7 : volume
10 : pan (left to right)
11 : expression (soft to loud)
64 : sustain
91 : reverb
93 : chorus
Parameters:

  • chan – The channel to use for audio playback.

  • ctrl – The control to modify, examples provided above.

  • val – The new value for the control. Always ranges 0 to 127.

generate(num_frames, num_channels)

Generates and returns frames. Should be called every frame.

Parameters:

  • num_frames – An integer number of frames to generate.

  • num_channels – Number of channels. Can be 1 (mono) or 2 (stereo)

Returns:

A tuple (output, True). The output is a numpy array of length (num_frames * num_channels)

noteoff(chan, key)

Stops a note.

Parameters:

  • chan – The channel on which the note should be stopped.

  • key – The key to stop.

noteon(chan, key, vel)

Plays a note.

Parameters:

  • chan – The channel to use for audio playback.

  • key – The midi key to play.

  • vel – The velocity to play the note at – correlates with volume. Ranges from 0 to 127.

pitch_bend(chan, val)

Adjust pitch for all notes of a channel. A pitch bend value of 0 is no pitch change from default. For the default setting of +/- 2 semitones: - A value of -4096 is 1 semitone down. - A value of 4096 is 1 semitone up. Maximum values are -8192 to +8192. See Synth.set_pitchbend_range() to adjust the semitone range.

Parameters:

  • chan – The channel to use for audio playback.

  • val – The value to adjust pitch by, as specified above.

program(chan, bank, preset)

Essentially defines which instrument this generator’s synth uses. Specific bank and preset combinations can be used to switch between different instruments.

Parameters:

  • chan – The channel to use for audio playback.

  • bank – The sound bank to use.

  • preset – The preset to use.

set_pitchbend_range(chan, semitones)

The default pitchbend range is +/- 2 semitones. Use this to set a new pitchbend range for the given channel.

Parameters:

  • chan – The channel this function controls.

  • semitones – The number of +/- semitones to set the pitchbend range.
